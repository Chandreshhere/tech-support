# Home Screen (Main Interface)

- Three-column layout:

## Left Sidebar (Server List)
- Narrow vertical bar (~72px wide)
- Top: Discord home button (Discord logo icon) — clicking it shows friends/DMs
- Below: list of server icons arranged vertically (circular icons)
- Between home and servers: horizontal separator line
- Bottom of sidebar: "Add a Server" button (green + icon), "Explore Public Servers" button (compass icon)

## Middle Panel (Channel List / DM List)
- Shows when a server or home is selected
- If home selected: shows "Find or start a conversation" search bar at top, then list of DM conversations
- If server selected: server name at top (clickable for server menu), below that channel categories with collapsible channel lists
- Bottom: user panel showing current user's avatar, username, status, and icons for microphone (mute), headphones (deafen), settings (gear icon)

## Right Panel (Content Area)
- Takes up most of the screen width
- Shows the content for the currently selected item (channel messages, DM chat, friends list, etc.)
- Has its own header bar at top showing the context (channel name, user name, etc.)
