# User Settings Screen

- Accessed by clicking the gear icon at bottom of the middle panel (next to username)
- Opens as a full-screen overlay with dark background

## Layout
- Left sidebar: settings categories list
- Right pane: settings content for selected category
- Top-right corner: "X" button (or Escape) to close settings

## Settings Categories (left sidebar)
- **User Settings section:**
  - My Account
  - Profiles
  - Privacy & Safety
  - Family Center
  - Authorized Apps
  - Devices
  - Connections
  - Clips
  - Friend Requests
- **Billing Settings section:**
  - Nitro
  - Server Boost
  - Subscriptions
  - Gift Inventory
- **App Settings section:**
  - Appearance
  - Accessibility
  - Voice & Video
  - Text & Images
  - Notifications
  - Keybinds
  - Language
  - Streamer Mode
  - Advanced
- **Bottom:**
  - Log Out (red text)
  - App version info
