# Server Settings Screen

- Accessed by clicking server name at top of channel list → "Server Settings" in dropdown
- Opens as a full-screen overlay (same style as user settings)

## Layout
- Left sidebar: server settings categories
- Right pane: settings content
- Top-right: "X" to close

## Settings Categories (left sidebar)
- **Server section:**
  - Overview
  - Roles
  - Emoji
  - Stickers
  - Soundboard
  - Widget
- **Community section (if enabled):**
  - Community Overview
  - Server Insights
  - Welcome Screen
- **Moderation section:**
  - Safety Setup
  - AutoMod
  - Audit Log
  - Bans
- **User Management:**
  - Members
  - Invites

## Overview Page (default)
- Server icon (editable, clickable to upload)
- Server name input field
- Server banner (Nitro required)
- System Messages Channel dropdown
- Inactive Channel dropdown
- "Delete Server" button at the very bottom (red, dangerous)
