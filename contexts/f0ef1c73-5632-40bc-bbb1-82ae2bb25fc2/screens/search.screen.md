# Search Screen

- Accessed by clicking the search icon in the top-right of the right panel header
- Opens a search popover/dropdown in the top-right area

## Search Input
- Text field with placeholder "Search"
- Supports search filters:
  - `from:` — filter by user
  - `mentions:` — filter by mentioned user
  - `has:` — filter by attachment type (link, embed, file, video, image, sound)
  - `before:` / `after:` / `during:` — date filters
  - `in:` — filter by channel

## Search Results
- Displays matching messages in a scrollable list below the search input
- Each result shows: channel name, user, timestamp, message snippet with highlighted matching text
- Clicking a result jumps to that message in context
- Top of results: result count and filter chips for active filters
