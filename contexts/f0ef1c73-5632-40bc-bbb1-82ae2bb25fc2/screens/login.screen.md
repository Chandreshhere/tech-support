# Login Screen

- Centered card on dark background
- Top: Discord logo + "Welcome back!" heading
- Two input fields stacked vertically:
  - Email or Phone Number (text input)
  - Password (password input, has eye icon to toggle visibility)
- Below password field: "Forgot your password?" link (blue text)
- Below that: "Log In" button (full width, purple/blue)
- Below login button: "Need an account? Register" link
- Bottom: QR code login option on the left side
