# Signup Screen

- Centered card on dark background
- Top: "Create an account" heading
- Form fields stacked vertically:
  - Email (text input)
  - Display Name (text input)
  - Username (text input)
  - Password (password input)
  - Date of Birth (three dropdowns: month, day, year)
- Checkbox: consent to terms of service
- "Continue" button (full width, purple/blue)
- Bottom: "Already have an account?" with "Log In" link
