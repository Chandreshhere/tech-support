# Friends List Screen

- Accessed via Discord home button in left sidebar, then "Friends" tab at top
- Top bar has tabs: Online, All, Pending, Blocked, Add Friend
- "Add Friend" button (green) also in the top bar on the right

## Friends List (when Online/All tab selected)
- Search bar at top: "Search" to filter friends
- List of friends, each row shows:
  - User avatar (left)
  - Username + status text (middle)
  - Action buttons on right: Message (chat bubble icon), Voice Call, Video Call, More (three dots)

## Add Friend Tab
- Input field: "You can add friends with their Discord username"
- "Send Friend Request" button (blue)

## Pending Tab
- List of incoming/outgoing friend requests
- Each row: avatar, username, accept (checkmark) / reject (X) buttons
