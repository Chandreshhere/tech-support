# Voice Channel Screen

- Accessed by clicking a voice channel in the channel list (preceded by speaker icon)
- Joining a voice channel changes the bottom of the middle panel

## Voice Connected Panel (bottom of middle panel)
- Shows: green "Voice Connected" indicator with channel name
- Buttons: disconnect (phone icon with X), camera toggle, screen share toggle
- Shows list of users currently in the voice channel with their avatars

## Voice Channel View (right panel, if selected)
- Grid/tile layout of participants
- Each tile shows: user avatar (or video feed if camera on), username, microphone status icon
- Bottom bar: mute, deafen, video, screen share, activities, disconnect buttons
- If someone is screen sharing: their stream takes up the main area, other participants shrink to a sidebar
