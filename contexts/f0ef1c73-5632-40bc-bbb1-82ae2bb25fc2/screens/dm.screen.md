# Direct Message Screen

- Accessed by clicking a DM conversation in the middle panel (when home is selected)
- Layout is nearly identical to text channel screen

## Header Bar (top)
- User avatar + username on the left
- Right side icons: voice call, video call, pinned messages, add friends to DM, members, search, inbox

## Message Area (center, scrollable)
- Same as text channel: chronological messages with avatars, timestamps, reactions
- Top of conversation shows: user avatar (large), username, "This is the beginning of your direct message history with @username"

## Message Input (bottom)
- Same as text channel input
- Placeholder: "Message @username"
