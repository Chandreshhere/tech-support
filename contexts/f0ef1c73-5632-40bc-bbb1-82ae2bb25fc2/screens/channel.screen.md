# Text Channel Screen

- Accessed by clicking a text channel name in the middle panel (preceded by # icon)

## Header Bar (top)
- Channel name with # icon on the left
- Channel topic/description text (if set)
- Right side icons: threads, pinned messages, members list toggle, search, inbox, help

## Message Area (center, scrollable)
- Messages displayed chronologically, newest at bottom
- Each message shows: user avatar, username, timestamp, message content
- Messages can contain: text, images, embeds, reactions, replies
- Hover on a message reveals action buttons: react, reply, thread, more
- Typing indicator at bottom when someone is typing

## Message Input (bottom)
- Text input field with placeholder "Message #channel-name"
- Left of input: "+" button to attach files
- Right of input: emoji picker, GIF, sticker, gift buttons
- Supports slash commands (type / to see list)
